# TicTacToeUnity
UNITY 井字棋游戏
1.两种游玩模式，人机模式和本地双人对战
2.支持本次游戏对局记录
3.支持切换AI对局难度

游戏流程：
1.由StartScene进入游戏开始界面，通过设置可设置背景音大小、对战难度、先执顺序
2.点击开始游戏进行对局，切换至MainScene对局场景
3.选择对局模式，每次对局结束会记录胜者与用时，步数数据，通过对局记录可查看。

3.可添加登录、注册功能，保存每个用户的单独对局记录，添加选择角色功能等。
